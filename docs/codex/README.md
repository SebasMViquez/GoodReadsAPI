# Codex Continuous Documentation

This folder is the permanent engineering log generated during collaboration with Codex.

## Files

- `WORKLOG.md`: every feature, fix, and structural change.
- `BUILD_LOG.md`: every compilation/test run and its result.
- `supabase/`: SQL and setup material for Supabase.

## Logging rule

Every coding session must append:

1. What changed.
2. Why it changed.
3. What was compiled/tested.
4. Exact result (`success`/`failed`) and key errors if any.
